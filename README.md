# NHK 風アナログ時計

テレビ放送でおなじみの **NHK 風の楕円フレーム付きアナログ時計** を、**HTML / CSS / JavaScript の単一ファイル**（`index.html`）で再現した個人用プロジェクトです。ビルドやパッケージマネージャは不要です。

> **注意**: 「NHK」は日本放送協会の商標です。本リポジトリは学習・個人利用を想定した非公式の再現であり、NHK 本体とは無関係です。

## 構成

| ファイル | 説明 |
|-----------|------|
| `index.html` | マークアップ・スタイル・針の制御・時報ロジックをすべて内包 |
| `jihou-sine-2f.mp3` | 時報用の短い音声（`index.html` と同じディレクトリに配置） |
| `.nojekyll` | GitHub Pages で Jekyll を無効化し、静的ファイルをそのまま配信しやすくする（空ファイルでよい） |

音声ファイルが無い場合でも時計表示は動きますが、時報は鳴りません。

## 使い方

1. リポジトリをクローンまたはダウンロードする。
2. `jihou-sine-2f.mp3` を `index.html` と同じフォルダに置く（任意だが時報に必要）。
3. `index.html` をブラウザで開く。

ローカルで `file://` として開いても動作します。CORS の都合で何か不具合が出る場合は、簡易の静的サーバー（例: `npx serve .`）で同じディレクトリを配信してください。

### GitHub Pages で公開する

リポジトリは [kzmanakzm-cpu/nhk](https://github.com/kzmanakzm-cpu/nhk) に置いています。GitHub Pages を有効にすると、次の URL で `index.html` が開きます（反映まで 1〜2 分かかることがあります）。

**公開 URL**: [https://kzmanakzm-cpu.github.io/nhk/](https://kzmanakzm-cpu.github.io/nhk/)

1. GitHub のリポジトリを開く → **Settings**（設定）
2. 左メニュー **Pages**
3. **Build and deployment** の **Source** で **Deploy from a branch** を選ぶ
4. **Branch** を `main`、フォルダを **`/ (root)`** にして **Save**

これで `main` ブランチのルートにある `index.html` がサイトのトップとして配信されます。音声は相対パス `jihou-sine-2f.mp3` のため、リポジトリに含まれていれば同じ Pages 上でも読み込まれます。

リポジトリ直下の **`.nojekyll`** は、GitHub Pages が Jekyll としてサイトを解釈するのを避け、静的ファイルをそのまま配信しやすくするための空ファイルです。

### ブラウザの自動再生ポリシー

多くの環境では、ユーザー操作の前に音声を自動再生できません。時報が鳴らないときは **画面を一度クリック（タップ）** すると再生が許可され、以降は時報が鳴る想定です。画面上部にその旨のヒントが出ることがあります。

## 機能の概要

- **見た目**: 青系の縦ストライプ背景、角丸の外枠、銀色のメタリック調の楕円ベゼル、黒い文字盤、12・3・6・9 の数字（SVG）、60 本分の目盛り（5 分ごとにドット）。
- **針**: 短針・長針は秒の小数まで反映して滑らかに回転。秒針は **1 秒ごとのカチッとした進み**に加え、わずかに **行き過ぎてから戻る** ような動き（CSS `transition` と整数秒の切り替えで実現）。
- **時報**: **5 分ごとの `:00` のちょうど 3 秒前**（分が `4, 9, 14, …` で秒が `57` のとき）に `jihou-sine-2f.mp3` を再生。同じタイミングでの二重再生を避けるフラグ処理あり。

## 挙動の調整（開発者向け）

`index.html` 内のスクリプトで、秒針の「ブレ」の強さなどを変えられます。

- `secondOvershootDeg` … 行き過ぎる角度（度）
- `secondSettleMs` … 戻りアニメーションの時間（秒）

針の見た目やレイアウトは CSS の `--face-d` や `.hand` 周りの変数・サイズで調整しています。

## 技術スタック

- 標準の HTML5 / CSS3 / ES5 互換の素の JavaScript（モジュールバンドラなし）
- アニメーションループは `requestAnimationFrame`
